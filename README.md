# Twitter Media Downloader

## Usage

```console
$ twget -h
Usage of twget:
  -decremental
    	search tweets decrementally
  -incremental
    	search tweets incrementally
  -limit int
    	limit number of tweets to search (default 2147483647)
  -output string
    	path to output directory
  -since string
    	filter tweets since this date
  -until string
    	filter tweets until this date
```
